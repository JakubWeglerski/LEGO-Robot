# LEGO-Robot
This project was developed during one of my study courses. The main goal was to design and build a robot that climbs along a line using only LEGO blocks. Additionally, the robot is capable of detecting the end of the line at the bottom and the ceiling at the top.
The main control module of the robot is the Husarion CORE2 driver.
You can find my project here: [https://drive.google.com/robot](https://drive.google.com/file/d/1LeJXvlAE3gpKTVCMAmB0CtVtIrfMwhCH/view).
